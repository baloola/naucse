Planned courses

1st Python Data Science course

We are organizing a 1st run of online Python Data Science course in English starting in April 2022, running for 13 weeks. It is expected that the attendees already have knowledge of Python basics. If you are not sure about your level, you can get in touch with us via email. You can find the full curriculum in the course details. The course will be run for two hours each week in a virtual classroom via Gather for around 15 attendees and several mentors.

If you are interested to join as an attendee, we will open the registration form during February 2022. We usually receive more applications than there are free slots. In that case we will need to select participants based on the answers from the form. If you would like to join as a side mentor, please send us email to pyladies.vienna@gmail.com.

4th Python beginners course - Autumn 2022

We are planning to open a 4th run of online Python programming beginners course in English starting in Autumn 2022, running for 13 weeks. It is aimed for complete beginners without any previous experience in programming. We will go through basic Python syntax, data types and structures and we will work on two projects along the way. The course will be run for two hours each week in a virtual classroom via Gather for around 15 attendees and several mentors.

Who are PyLadies?

PyLadies is an international mentorship group with a focus on helping more women become active participants and leaders in the Python open-source community. Our mission is to promote, educate and advance a diverse Python community through outreach, education, conferences, events and social gatherings.

For whom?

Our most important activity in PyLadies Vienna are online semestral Python beginners courses. They will lead you through complete Python programming fundamentals in an entertaining way. No boring math, all lessons are taught on easily imaginable examples and even programming of simple games.

What do you need?

In order to start, you need only motivation, enough free time and your own computer on which you can start to learn programming - with permission to install new software.
PyLadies Vienna are a part of international PyLadies.
